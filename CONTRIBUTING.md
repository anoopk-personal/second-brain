# Contributing

This is a personal project shared as a reference implementation. It is not actively seeking contributors, but feedback is welcome.

## Issues

Open an issue for:

- Bugs in the setup guide (broken steps, missing context, wrong commands)
- Security concerns (or use the [private reporting process](SECURITY.md))
- Suggestions for improvement

## Pull Requests

Please open an issue first to discuss the change before submitting a PR. This avoids wasted effort on changes that may not align with the project's direction.

## Code of Conduct

Be respectful and constructive. That is the entire policy.
